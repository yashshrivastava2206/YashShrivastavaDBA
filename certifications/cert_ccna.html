About This Certification

The Cisco CCNA 200-301 certification preparation course from LinkedIn Learning provides comprehensive training in network fundamentals and infrastructure access. This course covers essential networking concepts required for the Cisco Certified Network Associate (CCNA) certification, focusing on network architecture, protocols, and Cisco device configuration.

As a Database Administrator managing systems across network infrastructure, understanding networking fundamentals is crucial for troubleshooting connectivity issues, optimizing database communication, configuring secure connections, and ensuring high availability. This knowledge directly impacts database performance and reliability in distributed environments.

# IPv4 Subnetting Example
Network: 192.168.10.0/24
Subnet Mask: 255.255.255.0

Creating 4 subnets:
Subnet 1: 192.168.10.0/26 (Hosts: 192.168.10.1-62)
Subnet 2: 192.168.10.64/26 (Hosts: 192.168.10.65-126)
Subnet 3: 192.168.10.128/26 (Hosts: 192.168.10.129-190)
Subnet 4: 192.168.10.192/26 (Hosts: 192.168.10.193-254)

Each subnet provides 62 usable host addresses

! Basic Cisco Router Configuration
Router> enable
Router# configure terminal

MainRouter(config)# interface GigabitEthernet0/0

MainRouter(config-if)# no shutdown
MainRouter(config-if)# exit

! Static route example

! Access Control List (ACL) Example
Router(config)# access-list 10 permit 192.168.1.0 0.0.0.255
Router(config)# access-list 10 deny any
Router(config)# interface GigabitEthernet0/1
Router(config-if)# ip access-group 10 in

! Port Security on Switch
Switch(config)# interface FastEthernet0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport port-security
Switch(config-if)# switchport port-security maximum 2
Switch(config-if)# switchport port-security violation restrict
